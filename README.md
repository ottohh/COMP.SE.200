### Setting up local environment

1. **Install npm packages**
      ```
      npm install
   ```
2. **Run tests**
      ```
      npm run test
   ```
3. **Run coverage reporting**
      ```
      npm run coverage
   ```
       Coverage report appears in coverage/lcov-report/src/index.html
